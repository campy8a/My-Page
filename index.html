Data Analysis

Part of the benefits of creating BPMN models is, that you also create the data models of your business processes. Therefore, I went in deph as a business solutions engineer. And one of the resposabilities of this type of engineer is to have control over the efficiency of the models. That means to take track of the time, keep in track with the company's KPI. Why stop there, you can create Business Analytics on client's behavior, supplier's costs or times. And much more insights with all that data, at the palm of your hand. I've worked with tools such as Microsoft PowerBI, AWS Quicksight, and Tableau to achieve this goal.

SQL Programming

Since we create automated business processes thanks to our BPM models and automation tools, we have the data structures of each of the processes. This data need to be stored somewhere, either a S3 Server,mySQl server or SQL server to name a few. In order to create the data analysis of our processes, we need to lean how to consume that data. Usually the data is stored in SQL servers, therefore I developed a proeficiency in this language.
